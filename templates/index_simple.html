⬡ Publish-Subscribe (MQTT)

Model Asinkron - Publisher langsung return

Komponen Sistem (PS / MQTT)

  • MQTT Client A: Publisher (pengirim publish)
  • MQTT Broker (Mosquitto): routing berdasarkan topik
  • MQTT Client B/C/D: Subscriber (penerima broadcast)
  • Topik: chat/general
